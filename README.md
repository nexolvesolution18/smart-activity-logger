# smart-activity-logger
A CLI- based activity logger with time stamp and file storage using python
# FEATURES 
* Add activity logs with timestamp
* View all logs
* Search logs by keyword
* Count total logs
* presistent storage using JSON
# Stack 
* Python
* datetime
* json
# How to Run 
Run the script and follow menu options
